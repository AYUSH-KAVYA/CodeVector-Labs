# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-06-26
- Initial release of ProductDB demo.
- High performance FastAPI backend with asyncpg.
- Fast psycopg seeder logic (200k records).
- Keyset cursor-based pagination query implementation.
- Composite index optimization for query stability.
- Responsive React frontend with dark theme.
- Top right header actions for simulate inserts and manual refreshes.
